# RMDB (React Movie Database)

Full stack movie database website showing movie information along with features for a user to track and rate movies.

## Architecture

**Frontend**

- Languages:

  - Javascript
  - HTML
  - CSS

- Framework/Libraries:
  - React
  - Styled Components

**Backend**

- Language

  - Python

- Framework/Libraries

  - Django
    - Django Rest Framework
    - Django Rest Auth
    - Simple JWT

- Database:
  - SQLite
